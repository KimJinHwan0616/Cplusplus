># 포인터 - 배열
>포인터: 1번째 배열 값 주소
>
>### 주소, 변수
```
int arr[] = {1, 2, 3, 4, 5};
int *pa = arr;    // *(pa + 0) = arr + 0
```


## 주소 *(n번째 값)*
`배열 + n = 포인터 + n = &배열[n]`
```angular2html
arr = pa = &arr[0]    // 1번째 값 주소
arr + 2 = pa + 2 = &arr[2]    // 3번째 값 주소
```

## 변수 = 값
`배열[n] = *(배열 + n) = *(포인터 + n) = 포인터[n]`
```angular2html
arr[1] = *(ary + 1) = *(pa + 1) = pa[1]    // 2
```

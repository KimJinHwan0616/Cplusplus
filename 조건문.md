># 조건문 *(Conditional Statement)*
>`1`: 참, `0`: 거짓
>### 일반, 정수, 축약
논리 연산자: `&&`, `||`, `!`

---

## 일반
```angular2html
if ( 참＆거짓_표현식 )
{
   실행문;
}

else if ( 참＆거짓_표현식 )
{
   실행문;
}

    ...

else
{
   실행문;
}
```

## 정수
```
swich ( 정수_표현식 )
{
case 정수_표현식1:      // 정수_표현식 == 정수_표현식1
    실행문;
    break;

    ...
    
case 정수_표현식n:      // 정수_표현식 == 정수_표현식n
    실행문;
    break;

default:
    실행문;
    break;
}
```
```
int rank=2, m;

swich ( rank )
{
case 1:      // rank == 1
    m = 100;
    break;

case 2:      // rank == 2
    m = 200;
    break;

default:
    m = 10;
    break;
}
```

## 축약
`( 참＆거짓_표현식 ) ? 참_실행문 : 거짓_실행문;`

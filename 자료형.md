># 자료형 *(Data Type)*
>정수, 실수, 문자, 문자열, 배열
>
>### 문자, 문자열
>### 배열
>### 변환, 크기
정수: `int`, 실수: `double`

문자열: `string` *(using namespace std;)*

---

## 문자
`'문자 1개'`
```
'A', 'c', '#', '&', ...
```

## 문자열
`"문자 2개 이상"`
```
"안녕하세요", "Hello", ... 
```

---

## 변환 *(정수, 실수)*
`(숫자_자료형)숫자;`, `숫자_자료형(숫자);`★
```angular2html
(int)7.4;    // 7
(double)10;    // 10.0

int(7.4);    // 7
double(10);    // 10.0 
```

## 크기 *(byte)*
`sizeof(자료형)`
```
sizeof(7.4);    // 8
sizeof(int);    // 4
